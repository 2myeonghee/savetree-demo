# TreeMap 360 — 세이브트리 수목 관리 데모

by SaveTree × Greehill

## 데모
🔗 https://aiconsilium-dev.github.io/savetree-demo/

## 기능
- **3컬럼 다방식 레이아웃**: 좌측 수목 리스트 | 중앙 상세 정보 | 우측 트리맵
- **상단 필터바**: 건강등급/수종/센서상태 드롭다운 필터
- **수목 상세 5탭**: 기초 / 규격 / 위험성 / 생태계서비스 / 간섭정보
- **현황 사진**: 전경/수피/수관/근부 4종 촬영 영역
- **센서 현황**: 토양수분/pH/생체전기/EC 실시간 모니터링
- **Greehill STI 연동**: 수목 건강 지수 표시
